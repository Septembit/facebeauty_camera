# Face beauty camera
This project uses [face tools](https://github.com/mrronjt/face-toolbox-keras) as a tool to detect face mask, and uses guided filter to remove noise.

## Requirements

 - tensorflow-gpu==1.14
 - keras>=2.2.4
 - opencv-python>=3.4
